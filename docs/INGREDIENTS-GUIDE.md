# Cooking Ingredients Guide

## How to Find Cooking Ingredients in Recipe Builder

With the new **searchable dropdown**, you can easily find any ingredient by typing in the search box!

## Available Cooking Ingredients

### Cooking Oils & Fats
Search for: `"oil"`, `"ghee"`, `"butter"`

| Ingredient | Category | Calories per 100g |
|------------|----------|-------------------|
| Vegetable Oil | oils | 884 |
| Mustard Oil | oils | 884 |
| Olive Oil | oils | 884 |
| Coconut Oil | oils | 862 |
| Sesame Oil | oils | 884 |
| Ghee | dairy | 900 |
| Butter | dairy | 717 |

**Typical Usage in Recipes:**
- For cooking: 5-15g (1-3 tsp)
- For tempering: 5g (1 tsp)
- For sautéing: 10-20g (2-4 tsp)

## Tips for Recipe Building

### 1. Use Realistic Quantities
- **Oils/Fats:** 5-30g per recipe (not per serving!)
- **Spices:** 1-5g each
- **Salt:** 2-10g per recipe
- **Sauces:** 10-50g depending on type

### 2. Common Recipe Examples

**Simple Curry Base (for 4 servings):**
- Oil: 15g
- Onion: 100g
- Tomato: 100g
- Ginger: 10g
- Garlic: 10g
- Turmeric: 2g
- Red Chili Powder: 3g
- Coriander Powder: 5g
- Cumin Powder: 3g
- Salt: 6g

**Stir-Fry Base (for 2 servings):**
- Vegetable/Sesame Oil: 10g
- Garlic: 8g
- Ginger: 8g
- Soy Sauce: 15g
- Oyster Sauce: 10g

**Pasta Sauce (for 3 servings):**
- Olive Oil: 15g
- Garlic: 10g
- Tomato Sauce: 200g
- Salt: 4g
- Herbs/Spices: 2-3g each

### 3. Search Tips
- **Type partial names**: "oni" finds onion, "chick" finds chicken & chickpeas
- **Type category**: Search "protein" to see all protein sources
- **Type food type**: Search "veg" to see vegetarian options
- **Combine terms**: The search looks through multiple fields

### 4. Building Complete Recipes
A complete recipe typically includes:
1. **Base ingredients** (rice, pasta, bread)
2. **Protein sources** (chicken, paneer, eggs, dal)
3. **Vegetables** (2-3 different types)
4. **Cooking fat** (oil, ghee, butter)
5. **Aromatics** (onion, garlic, ginger)
6. **Spices** (turmeric, cumin, chili, etc.)
7. **Salt** (don't forget!)

## Example: Building a "Chicken Curry" Recipe

1. Search `"chicken"` → Add Chicken Breast (cooked) - 300g
2. Search `"oil"` → Add Vegetable Oil - 15g
3. Search `"onion"` → Add Onion - 100g
4. Search `"tomato"` → Add Tomato - 100g
5. Search `"ginger"` → Add Ginger (fresh) - 10g
6. Search `"garlic"` → Add Garlic (fresh) - 10g
7. Search `"turmeric"` → Add Turmeric Powder - 2g
8. Search `"chili"` → Add Red Chili Powder - 3g
9. Search `"coriander powder"` → Add Coriander Powder - 5g
10. Search `"salt"` → Add Salt - 6g

Set servings to 4, and the app automatically calculates:
- Total calories
- Calories per serving
- Total protein
- Protein per serving

2. Click **"Add Food Item"**
3. Enter the nutritional information
4. It will be immediately available for recipes!
