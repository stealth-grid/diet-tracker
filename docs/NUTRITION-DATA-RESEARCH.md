# Nutrition Data Research & Verification

This document details the comprehensive research and verification process for all food items in the Diet Tracker database.

## Data Sources

All nutritional values have been cross-referenced with:
1. **USDA FoodData Central** (primary source for international standards)
2. **Indian Food Composition Tables** (IFCT 2017)
3. **National Institute of Nutrition (NIN), India**
4. Published scientific nutrition databases

## Methodology

- Values rounded to 1 decimal place for protein, whole numbers for calories
- All values are per 100g serving
- Cooked items specified as "cooked" to avoid confusion
- Water absorption accounted for in cooked items
- Average values used for items with natural variation

## Notes for Users

### Important Considerations:
1. **Cooking Method Matters**: Cooked values assume no added fat unless specified
2. **Variation**: Natural foods can vary by variety, season, and growing conditions
3. **Preparation**: Adding oil, ghee, butter, or other ingredients increases calories
4. **Portions**: All values are per 100g - actual portions may differ
5. **Fresh vs Processed**: Values are for fresh/minimally processed foods

### Common Pitfalls to Avoid:
- ‚ùå Don't confuse raw and cooked values (especially for grains and legumes)
- ‚ùå Don't forget to account for added fats in cooking
- ‚ùå Don't use "average" fish values for fatty fish like salmon
- ‚ùå Don't assume soaking removes calories (it only adds water weight)
